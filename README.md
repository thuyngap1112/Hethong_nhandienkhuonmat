### Description
```
App desktop attendance use cv2
```

### Environment
```
- mysql: 8.0.19
- python: 3.10.10
```

### Pip install
```
$ pip install imagehash
$ pip install Pillow
$ pip install pywin32
$ pip install pymysql
$ pip install xlwt
$ pip install opencv-contrib-python
$ pip install opencv-python
```

### Database
```
- open file attendance.sql
```

### Run App
```
cd folder src => python start_gui_app.py
```

### Exit realtime attendance
```
enter key esc
```
